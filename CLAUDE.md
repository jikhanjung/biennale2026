# CLAUDE.md

2026 미학비엔날레 홍보 웹사이트. 자세한 내용은 `README.md` 참고.

## 프로젝트 성격

- **빌드/의존성 없는 정적 사이트.** 번들러·프레임워크·패키지 매니저 없음. HTML 파일을 그대로 브라우저로 연다.
- 각 `indexN.html`은 **단일 파일에 CSS·JS가 모두 인라인**되어 있다. 외부 의존성은 `index1.html`의 Google Fonts CDN뿐.
- 원본 자료는 모두 `data/`에 있다. 콘텐츠의 단일 출처(source of truth)는 `data/`의 이미지/PDF다.

## 파일 구조

- `index1.html` — 손으로 디자인한 페이지. 텍스트는 `data/` 이미지에서 옮겨 적은 것.
- `index2.html` — 이미지 12장 세로 스크롤. **텍스트·장식 일절 없이 이미지 원본만** 보여주는 것이 의도이므로 요소를 추가하지 말 것.
- `index3.html` — 스크롤 구동 크로스페이드. 조절 손잡이 두 개:
  - `W` (update 함수 내) — 크로스페이드 폭. 작을수록 빠르고 칼같이 전환. 현재 `0.10`.
  - `VH_PER_IMAGE` — 이미지당 스크롤 길이(vh). 작을수록 빨리 넘어감. 현재 `100`.
- `data/KakaoTalk_*_01.png ~ _12.png` — 프로그램 북 12쪽. `_01`이 메인 포스터.
- `data/미학비엔날레2026_프로그램.pdf` — 같은 내용의 PDF. **텍스트 레이어가 없는 이미지 기반 PDF**라 `pdftotext`로는 내용이 추출되지 않는다. 내용 확인은 PNG를 직접 읽어야 한다.

## 작업 시 유의

- 디자인 톤: 포스터를 따른다. 핑크 그라데이션 배경 + 빨강(`#e5342a`) 스탬프 슬로건, 신체적(뇌–혀–심장) 모티프. **무거운 디스플레이 폰트/`font-weight:900`은 쓰지 않는다** — 가독성이 떨어진다. 제목은 Noto Sans KR 700 정도로.
- 행사 핵심 정보(일시/장소/주최/라인업)는 `README.md`에 정리돼 있다. 사실 변경 시 README와 해당 HTML을 함께 갱신.
- 이미지 파일명은 길고 한글 PDF명이 포함되므로 셸에서 다룰 때 인용/이스케이프에 주의.
